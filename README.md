# 🎯 AutoStrike: Intelligent Object Tracking & Targeting System

An advanced **real-time object tracking and automated targeting system** built using computer vision and embedded control. AutoStrike uses a camera to detect objects and dynamically controls servo motors to track and align with the target.

---

## 📌 Overview

AutoStrike is a smart surveillance and tracking prototype that combines **computer vision (OpenCV)** with **hardware control (Raspberry Pi + Servo Motors)**. The system detects objects in real time, calculates their position, and moves a mechanical system to follow or target them.

---

## ⚙️ Features

* 🎥 Real-time object detection using OpenCV
* 🎯 Automated target tracking system
* 🔄 Dual servo motor control for precise movement
* 🧠 Intelligent coordinate-based positioning
* ⚡ Embedded system using Raspberry Pi
* 🔌 Efficient power regulation using LM2596

---

## 🧰 Tech Stack

**Software:**

* Python 3
* OpenCV
* NumPy

**Hardware:**

* Raspberry Pi
* Camera Module
* 2x Servo Motors (MG996R)
* LM2596 Buck Converter
* Power Supply (Lithium-ion Battery)

---

## 🔌 Hardware Architecture

The system consists of:

* Camera capturing real-time video
* Raspberry Pi processing frames
* Servo motors controlling directional movement
* Power module regulating voltage supply

---

## 🧭 Working Principle

1. 📷 Camera captures live video feed
2. 🧠 OpenCV processes frames and detects objects
3. 📍 Object position (coordinates) is calculated
4. 🔄 Control signals sent to servo motors
5. 🎯 System aligns itself with the detected object
